Pattaya leads (Google Places API)

Text Search runs on the server only — your Google API key is never sent to the browser. Open this page from the same origin as the Node server (e.g. http://localhost:8080/places-leads.html). Each line runs POST /api/places/search. Optional per-line override: clinic|your query here, hotel|…, or real_estate|….

Results are biased toward Pattaya / Thailand on the server (region + map circle). Up to 60 places per query (paginated server-side).
